Bookmarklets

Bookmarklets are browser bookmarks that instead of going to a http:// URL, run some JavaScript on the current page. Bookmarklets are usually placed on the browser's bookmark toolbar so they are always visible. They don't get favicons or any styling. They work on mobile devices.

Browser Extensions

Most browser extensions are just bookmarklets that get a permanent place in the browser's UI. They are signed by the developer and installed from the browser vendor's site to avoid security warnings.

Clicking a the browser extension button would have the same effect as clicking a bookmarklet button (unless we spent days making them).

The bookmarklets

There are 3 main ways bookmarklets can work. Drag the 3 proof-of-concept bookmarklets below to your bookmark bar to try them.

  • Add to GC (iframe) - load a thickbox-style iframe over the current page.
  • Add to GC (redir) - redirect from the current page to Gift Circle
  • Add to GC (popup) - open a popup window with Gift Circle in

Don't see the Bookmarks toolbar in your web browser?

Did you drag the button up there?

Ready for the fun part?
